Sift — Alert Triage & Learning System

          

            A self-hosted triage engine for drowning SOCs. Every alert gets an itemized
            receipt — which signals fired, the points they carried, and plain-English
            reasoning — so an analyst can agree or override in seconds. It tracks those verdicts
            and statistically recalibrates noisy rules over time. Ingests eight formats
            (Wazuh, Suricata, Elastic, AWS GuardDuty, M365 Defender, CrowdStrike, osquery,
            generic webhooks), correlates related alerts into cases, and pulls free threat-intel
            feeds with zero API keys.
